String Operations & Casting Trace Table Challenges

These challenges involve various string operations and type‐casting tasks set in the context of an English high school. Fill in the trace table only when a variable’s value changes or output is produced.

ST1: String Length

Scenario: A student is preparing a project poster for their English class and needs to know the length of the course title "EnglishLiterature".

Pseudocode:

01 subject = "EnglishLiterature"
02 length = subject.length
03 print("Length is " + length)
Line Number subject length Output

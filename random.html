Random Numbers Trace Table Challenges

Each challenge uses a random number function. Complete the trace table by entering only the value for each line. (For example, if the code assigns “roll = random(1,6)” and the random number is 4, simply enter “4”.)

Q1: Dice Roll Simulation

Scenario: A school security system simulates a dice roll to generate a code.

Pseudocode:

01 roll = random(1,6)
02 print("You rolled " + roll)

Trace Table:

Line Number Value
01 (roll)
02 (Output)

Q5: Loop Iteration Count

Scenario: A simulation runs a loop a random number of times and prints the iteration count each time, then reports the total number of iterations.

Pseudocode:

01 count = 0
02 n = random(1,5)
03 do
04    count = count + 1
05    print("Iteration " + count)
06 while (count < n)
07 print("Loop finished after " + count + " iterations")

Trace Table:

Line Number Value
02 (n)
04 (Iteration 1)
04 (Iteration 2)
04 (Iteration 3)
07 (Output)

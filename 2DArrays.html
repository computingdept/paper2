2D Array Trace Table Challenges

Below are 10 trace table challenges involving 2D arrays in various high school scenarios. Complete the table only when a variable’s value changes or when output is produced.

Q1: Basic Array Access

Scenario: In the art class, a teacher uses a digital palette stored in a 2D array representing a game board of colours. The palette is defined as: ["Blue", "Pink", "Green", "Yellow", "Red"] Copy and paste original array values from here -> Blue, Pink, Green, Yellow, Red (only one row). The program prints the colour at index 2.

Pseudocode:

01 colours = ["Blue", "Pink", "Green", "Yellow", "Red"]
02 print(colours[2])
Line Number colours Output

Q2: Array Element Modification

Scenario: In a sports analysis, a student has an array of scores from a recent match: [65, 70, 75, 80, 85]. The score at index 3 is updated from 80 to 90. The program then prints the updated array.

Pseudocode:

01 scores = [65,70,75,80,85]
02 scores[3] = 90
03 print(scores)
Line Number scores Output

Q3: Sum of Array Elements

Scenario: A student calculates total marks from an exam where marks are stored in a 2D array representing a single row: [10,20,30,40,50].

Pseudocode:

01 marks = [10,20,30,40,50]
02 total = 0
03 for i = 0 to 4
04     total = total + marks[i]
05 endfor
06 print("Total marks: " + total)
Line Number i total Output

Q4: Finding Maximum in an Array

Scenario: A teacher records test scores in a 2D array (3×3 matrix): [[55,78,82],[67,90,73],[80,85,88]] and wants to determine the highest score.

Pseudocode:

01 tests = [[55,78,82],[67,90,73],[80,85,88]]
02 max = tests[0,0]
03 for i = 0 to 2
04   for j = 0 to 2
05     if tests[i,j] > max then
06         max = tests[i,j]
07     endif
08   endfor
09 endfor
10 print("Max score is " + max)
Line Number i max Output

Q5: Array Element Search

Scenario: A classroom seating chart is stored in a 2D array: [["Bob","Carol","Dave"],["Eve","Alice","Frank"],["Grace","Heidi","Ivan"]]. The program searches for "apple" (note: here we search for "Alice") and prints her seat position.

Pseudocode:

01 seats = [["Bob","Carol","Dave"],["Eve","Alice","Frank"],["Grace","Heidi","Ivan"]]
02 found = false
03 i = 0
04 while (i < 3 and found == false)
05   j = 0
06   while (j < 3 and found == false)
07     if seats[i,j] == "Alice" then
08         found = true
09         position = "(" + i + "," + j + ")"
10     endif
11     j = j + 1
12   endwhile
13   i = i + 1
14 endwhile
15 print("Alice found at " + position)
Line Number seats found Output

Q6: Count Occurrences of Empty Cells

Scenario: In a board game, an 3×3 game board is stored in a 2D array where empty cells are represented by "". For example: [["Pawn", "", "Pawn"],["", "Pawn", ""],["Pawn", "", ""]] The program counts the number of empty cells.

Pseudocode:

01 gameboard = [["Pawn","","Pawn"],["","Pawn",""],["Pawn","",""]]
02 count = 0
03 for i = 0 to 2
04   for j = 0 to 2
05     if gameboard[i,j] == "" then
06         count = count + 1
07     endif
08   endfor
09 endfor
10 print("Empty count: " + count)
Line Number gameboard count Output

Q7: Transpose of a Matrix

Scenario: In a maths project, a student is given a 2×3 matrix [[1,2,3],[4,5,6]] and needs to compute its transpose.

Pseudocode:

01 matrix = [[1,2,3],[4,5,6]]
02 transpose = new array[3,2]
03 for i = 0 to 1
04   for j = 0 to 2
05     transpose[j,i] = matrix[i,j]
06   endfor
07 endfor
08 print(transpose)
Line Number matrix / transpose Output

Q8: Array Row Update

Scenario: A classroom seating chart is stored in a 2D array: [["Bob","Carol","Dave"],["Eve","Alice","Frank"],["Grace","Heidi","Ivan"]]. A student changes the entire second row to: ["Zoe","Liam","Mia"] and then prints the updated chart.

01 seats = [["Bob","Carol","Dave"],["Eve","Alice","Frank"],["Grace","Heidi","Ivan"]]
02 seats[1] = ["Zoe","Liam","Mia"]

Q9: Extract Subarray

Scenario: A 4×4 game map is stored in a 2D array. A student extracts the top‑left 2×2 subarray for analysis.

Pseudocode:

01 map = [[1,2,3,4],

02 submap = new array[2,2]
03 for i = 0 to 1
04   for j = 0 to 1
05     submap[i,j] = map[i,j]
06   endfor
07 endfor
08 print(submap)
Line Number map / submap Output

Q10: Concatenation from a Letter Grid

Scenario: In a word puzzle, a 2D array of letters is given: [["H","E","L","L","O"],["W","O","R","L","D"],["T","E","S","T","S"]]. A student concatenates all the letters in the second row to form a word.

Pseudocode:

01 letters = [["H","E","L","L","O"],
             ["W","O","R","L","D"],
             ["T","E","S","T","S"]]
02 word = ""
03 for j = 0 to 4
04    word = word + letters[1,j]
05 endfor
06 print("Word is " + word)
Line Number letters / word Output

Sequence Trace Table Challenges

These challenges show how a SELECTION statement does NOT run in a strict sequence.

Task 1: Shop Discount – In a local shop, a customer spends £70 and is a member. What discount do they get?

Pseudocode:

01 purchaseAmount = input("Enter purchase amount")
02 memberStatus = input("Enter member status (yes/no)")
03 if purchaseAmount >= 100 then
04   discount = 20
05 elseif purchaseAmount >= 50 then
06   discount = 10
07 else
08   discount = 0
09 endif
10 if memberStatus == "yes" then
11   discount = discount + 5
12 endif
13 print("The discount is" + discount)
Line Number purchaseAmount memberStatus discount Output

Task 10: Speeding Fine – A car is travelling at 80 in a 60 zone. What is the fine?

Pseudocode:

02 speedLimit = input("Enter speed limit")

04   fine = (speed - speedLimit) * 5
05 else
06   fine = 0
07 endif
08 print("The speeding fine is " + fine)

Q1: Linear Search – Find a Name

Scenario: In a classroom, an array of student names is stored as ["Bob", "Carol", "Alice", "David", "Eve"]. A program searches for "Alice".

Pseudocode:

01 names = ["Bob", "Carol", "Alice", "David", "Eve"]
02 i = 0
03 while (i < 5)
04     if names[i] == "Alice" then
05         print("Found at index " + i)
06         break
07     endif
08     i = i + 1
09 endwhile

Q4: Binary Search – Number Found

Scenario: A sorted array of numbers [2,5,7,11,13,17,19] is searched for 11 using binary search.

Pseudocode:

01 numbers = [2,5,7,11,13,17,19]
02 low = 0
03 high = 6
04 while (low <= high)
05     mid = (low + high) // 2
06     if numbers[mid] == 11 then
07         print("Found 11 at index " + mid)
08         break
09     elseif numbers[mid] < 11 then
10         low = mid + 1
11     else
12         high = mid - 1
13     endif
14 endwhile

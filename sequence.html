Sequence Trace Table Challenges

These challenges show how a programme runs line by line in SEQUENCE—without any SELECTION (like “if” or switch/case) or ITERATION (for/while loops). Each line happens in order, and we keep track of the values and outputs at each step.

At a small café, customers use a machine to pay for their food. One customer enters 5 for one item and 3 for another.

Pseudocode:

01 a = int(input("First number"))
02 b = int(input("Second number"))
03 sum = a + b
04 print("Sum is " + sum)

Record the trace only when a value changes or output is produced.

Line Number a b sum Output

Example 6: Sharing Pencils – A class has 17 pencils that are shared equally among 5 pupils. How many pencils does each get and how many are left?

Pseudocode:

01 a = int(input("First number"))
02 b = int(input("Second number"))
03 modResult = a MOD b
04 divResult = a DIV b
05 result = (a - b) * modResult + divResult
06 print("Result is " + result)

Record the changes.

Line Number a b modResult divResult result Output
